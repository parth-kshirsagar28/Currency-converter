# 💱 Currency Converter

A real-time currency converter that fetches exchange rates using a public Currency API. Users can convert between different currencies with automatically updated exchange rates.

## 🚀 Features

- Live exchange rates
- Select different currencies
- Country flags
- User-friendly interface
- Real-time conversion

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Currency API
